# Neovim設定 基本設計書

## 1. 全体設計

### 1.1. プラグイン一覧
採用する**プラグイン**とその**選定理由**、**役割**を記述します。
**※WIP※**

| プラグイン名 | 役割 | 選定理由 | 依存関係 |
| :--- | :--- | :--- | :--- |
| `lazy.nvim` | プラグインマネージャー | 高速なロード、遅延ロード機能、設定のモジュール化をサポートするため。 | なし |
| `tokyonight.nvim` | カラースキーム | 視認性が高く、目に優しい配色であるため。 | なし |
| `nvim-treesitter` | 構文ハイライト・構造化 | 高精度な構文解析とテキストオブジェクトを提供するため。 | なし |
| `nvim-lspconfig` | LSPクライアント | 複数の言語サーバーを統一的に管理するため。 | `nvim-cmp` |


## 2. 詳細設計

### 2.1. 基本設定 (`lua/core/options.lua`)
**※WIP※**

| オプション名 | 設定値 | 目的 |
| :--- | :--- | :--- |
| `opt.number` | `true` | 行番号を表示する。 |
| `opt.relativenumber` | `true` | 相対行番号を表示し、移動を容易にする。 |
| `opt.tabstop` | `4` | タブ文字の幅を4スペースにする。 |
| `opt.shiftwidth` | `4` | 自動インデントの幅を4スペースにする。 |
| `opt.expandtab` | `true` | タブをスペースに展開する。 |

### 2.2. キーマップ (`lua/core/keymaps.lua`)
**※WIP※**

| モード | キー | 実行コマンド/関数 | 目的 |
| :--- | :--- | :--- | :--- |
| ノーマル | `<leader>pv` | `:Ex<CR>` | Netrw (ファイルエクスプローラー) を開く。 |
| ノーマル | `<leader>ff` | `:Telescope find_files<CR>` | プロジェクト内のファイルをファジー検索する。 |
| ノーマル | `<leader>ws` | `<cmd>w<CR>` | ファイルを保存する。 |
| ノーマル | `gd` | `<cmd>lua vim.lsp.buf.definition()<CR>` | 定義元へジャンプする。 |
